## Overview

This file defines what an AI copilot should follow when works on an Issue assigned to it in GitHub. The instructions here should be executed on top on the defined in the Issue.

## Objectives

- Increase with 1 the PATCH part from the [https://semver.org/](https://semver.org/) style of the version file (the empty file with format vMAJOR.MINOR.PATCH (Example: v1.0.2) in folder `.aib_brain/`
- Add entry in `logs\versions_log.md` describing the changes accordingy the current Issue following the existing formatting. Only append, do not modify the existing content

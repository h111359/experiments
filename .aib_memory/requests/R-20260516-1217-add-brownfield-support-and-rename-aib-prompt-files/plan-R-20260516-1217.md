## Goal

Rename `.aib_brain/prompts/aib-analysis.md` to `aib-analyze.md` and `.aib_brain/prompts/aib-context.md` to `aib-refresh-context.md` to follow action-verb naming consistent with `aib-implement.md`, update all workspace references to reflect the new names, add a brownfield pre-preflight step to `aib-analyze.md` that auto-executes the context generation workflow when `context.md` is absent or empty, and update the `menu.py` advisory message to state that context gathering will execute automatically before analysis.

## Constraints

- Resolved filenames: `aib-analyze.md` (analysis prompt) and `aib-refresh-context.md` (context prompt).
- `.aib_brain/` files may be modified when the active request explicitly authorises the change (as this request does).
- `context.md` must not be manually edited; it must be regenerated by executing `aib-refresh-context.md` after all renames are complete.
- Tool scripts (`.aib_brain/tools/*.py`) other than `menu.py` must not be modified by this request.
- All test files must pass without modification of test logic — only path constants and string literals referencing old filenames may be updated.
- The brownfield detection step must not create a recursive loop; `aib-refresh-context.md` must not invoke `aib-analyze.md`.
- Python 3.10+ standard library only for any script changes (`menu.py`).
- The `analysis-convention.md`, `plan-convention.md`, and `requirements-analysis-convention.md` files must not be modified.

## Success criteria

- The renamed analysis prompt file exists at `.aib_brain/prompts/aib-analyze.md` and the old `aib-analysis.md` no longer exists.
- The renamed context prompt file exists at `.aib_brain/prompts/aib-refresh-context.md` and the old `aib-context.md` no longer exists.
- `aib-analyze.md` contains a pre-preflight brownfield detection step (before Preflight Step 1) that checks for empty/absent `context.md` and instructs the AI to execute `aib-refresh-context.md` before proceeding.
- `menu.py` advisory message for empty `context.md` reads that context gathering will execute automatically before analysis and references `aib-refresh-context.md`.
- `tests/test_analysis_prompt_structure.py`, `tests/test_context_formatting_rules.py`, `tests/test_instructions_md.py`, and `tests/test_menu.py` all pass with no `FileNotFoundError` or assertion failures related to old filenames.
- `.aib_memory/context.md` is regenerated and reflects the new prompt filenames throughout.
- `.aib_brain/README.md` references only the new filenames in its invocation instructions.

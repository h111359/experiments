## Goal

Perform a consistency audit of the workspace's documentation, conventions, templates, and prompts. Identify all issues including misalignments, inconsistencies, logical errors, redundancies, misplaced content, unclear wording, broken cross-references, and format drift. Produce a written analysis file (`consistency-analysis-[TIMESTAMP].md`) documenting every detected issue with evidence, explanation of why it matters, and concrete improvement proposals.

## Background

The AI_Builder repository hosts an AI-assisted development workflow built around a set of conventions, prompt files, and tooling scripts. Over multiple iterations, these artifacts have grown incrementally. Inconsistencies may have accumulated across prompts, conventions, and templates that could cause agent misbehavior, ambiguous instructions, or conflicting guidance. A systematic consistency audit will surface these issues so they can be addressed in follow-up requests.

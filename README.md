# AI Builder

## Overview

AI Builder (AIB) is a minimal, model-agnostic framework for specification-driven development.

## Installation

Download the versioned archive for your target release from the `versions/` folder in this repository:

1. Navigate to the `versions/` folder in the repository and download the latest or other available version.
2. Unzip the archive into the root of your workspace. The archive contains a `.aib_brain/` folder — place it directly at the workspace root.
3. Run the launcher for your operating system:
   - **Windows:** `.\.aib_brain\run.bat`
   - **Linux/macOS:** `.aib_brain/run.sh`

Follow the user guide in `.aib_brain/README.md`
